# SMART-_ECOMMERCE
ShopSmart E-commerce Analytics + GA4 + Ads (ML &amp; BI)  Solving poor marketing ROI, high churn, and unclear sales insights for an African e-commerce brand. Using Python (ML), SQL, and Tableau to predict churn, forecast CLV, optimize ad spend, and visualize multi-country performance across Kenya, Nigeria, South Africa, Ghana, and Egypt.
